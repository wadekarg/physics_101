Transistors & Logic Gates

How a tiny base current controls a large collector current — the amplifier and switch that built the digital world.

Theory

A Bipolar Junction Transistor (BJT) has three layers: Emitter, Base, and Collector. In an NPN transistor, a thin p-type base is sandwiched between n-type emitter and collector. A small current injected into the base (I_B) allows a much larger current to flow from collector to emitter (I_C). The ratio β = I_C / I_B is the current gain (typically 50–300). The three operating regions are: active (linear amplification, I_C = β × I_B), saturation (transistor fully ON, V_CE ≈ 0), and cutoff (transistor fully OFF, I_C ≈ 0).

As an amplifier, the transistor operates in the active region. A small AC signal superimposed on the DC base bias produces a magnified AC signal at the collector. The voltage gain is A_V = −β × R_C / r_e. As a switch, the transistor is driven between saturation (ON) and cutoff (OFF) by a digital signal. Every logic gate in every computer uses transistors as switches — and modern processors contain over 10 billion transistors on a chip the size of a fingernail.

Logic gates implement Boolean algebra using transistors. The NOT gate (inverter) uses a single transistor: HIGH input saturates the transistor, pulling output LOW; LOW input keeps it in cutoff, output stays HIGH. An AND gate puts two transistors in series (both must be ON for output to go HIGH). An OR gate puts them in parallel. NAND and NOR gates are universal — any logic function can be built using only NANDs or only NORs. All microprocessors, memory chips, and digital circuits are ultimately networks of NAND/NOR gates built from MOSFETs.

⚛️ About Physics 101

Physics 101 is a free, open-source, interactive learning platform covering the complete high-school and introductory university physics curriculum — built to make physics intuitive, visual, and actually fun.

🔓 Open Source  ·  MIT License  ·  Free Forever

📚 Educational Purpose

This platform is created purely for educational purposes. All content — explanations, formulas, simulations, quiz questions, and worked examples — is intended to help students understand physics concepts. It is not a substitute for a qualified teacher, official textbook, or examination board syllabus.

While every effort has been made to ensure accuracy, errors can slip through. If you find something incorrect, outdated, or misleading, please reach out — your correction helps every student who visits after you.

🛠️ Built With

  • Vanilla JavaScript (ES Modules) — no framework, no bundler
  • p5.js — canvas-based interactive simulations
  • Matter.js — 2D rigid-body physics engine
  • KaTeX — fast LaTeX formula rendering
  • Plain CSS with custom properties — no CSS framework
  • All progress stored locally in your browser — no account, no server
